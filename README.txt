PlayStation Cafe Manager V5 Fixed

طريقة التشغيل:
1) فك الضغط.
2) على الكمبيوتر افتح Start.bat.
3) على الموبايل افتح PlayStation_Cafe_Manager_V5.html باستخدام Chrome.

بيانات الدخول:
admin / 1234
cashier / 1234

ملاحظات:
- النسخة تعمل Offline بدون Supabase.
- Supabase اختياري فقط للمزامنة بين أكثر من جهاز.
- تم إصلاح مشكلة عدم فتح البرنامج بعد تسجيل الدخول ومشكلة شاشة المودال.

تفعيل Supabase:
1) افتح Supabase > SQL Editor.
2) انسخ محتوى supabase_schema.sql واضغط Run.
3) من Project Settings > API خذ Project URL و anon public key.
4) داخل البرنامج > الإعدادات/المزامنة، اختر Hybrid Supabase، ضع القيم، واضغط حفظ ثم مزامنة الآن.
